# Style Guide

초기 기본 스타일은 사용자가 별도 지시를 주지 않았을 때 적용하는 출발점입니다.

## 기본 방향

- 미니멀하고 현대적인 슬라이드 이미지
- 밝고 친근하지만 가볍게 보이지 않는 분위기
- 명확한 제목, 짧은 본문, 넉넉한 여백
- 한 장에 하나의 핵심 메시지
- 16:9 화면비

## 타이포그래피

- 제목은 크고 선명하게 둡니다.
- 본문은 1~3개의 짧은 문장이나 키워드 중심으로 정리합니다.
- 이미지 안 텍스트가 많아질 경우 의미를 유지하며 줄입니다.
- 필수 문구는 임의로 바꾸지 않습니다.

## 색상

- 배경은 밝고 읽기 쉬운 톤을 기본으로 합니다.
- 강조색은 한두 가지로 제한합니다.
- 슬라이드마다 색을 과도하게 바꾸지 않습니다.
- 사용자가 지정한 브랜드 색이나 금지 색상이 있으면 우선합니다.

## 레이아웃

- 표지, 전환, 결론 슬라이드는 더 과감한 구성을 사용할 수 있습니다.
- 정보 전달 슬라이드는 읽기 쉬운 구조를 우선합니다.
- 장식 요소보다 메시지 이해를 우선합니다.
- 슬라이드 간 여백, 글자 크기, 강조 방식이 일관되도록 합니다.

## 피해야 할 것

- 긴 문단을 그대로 이미지에 넣기
- 작은 글자에 많은 정보를 넣기
- 근거 없는 수치나 사실 추가
- 지나치게 복잡한 배경
- 슬라이드마다 완전히 다른 디자인 언어 사용
